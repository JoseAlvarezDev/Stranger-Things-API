API Documentation

Complete reference guide for the Stranger Things API. Access characters, episodes, locations, and more with simple REST endpoints.

Base URL

https://strangerthingsapi.com/api

All API requests should be made to this base URL. The API is completely open and requires no authentication.

Your First Request

Let's start by fetching a single character. Try this example:

GET /api/characters/1

Response Format

All responses are returned in JSON format with consistent structure:

{
  "id": 1,
  "name": "Eleven",
  "real_name": "Jane Hopper",
  "nickname": "El",
  "age": 15,
  "gender": "Female",
  "occupation": "Student",
  "portrayed_by": "Millie Bobby Brown",
  "powers": [
    "Telekinesis",
    "Remote Viewing",
    "Mind Control"
  ],
  "status": "Alive",
  "portrait_path": "/images/characters/eleven.webp",
  "quotes": [
    "Friends don't lie.",
    "Mouth-breather."
  ],
  "seasons": [1, 2, 3, 4]
}

Pro Tips
  • Images are served via CDN for optimal performance
  • Each page contains 20 items by default (max: 50)
  • Use filters to narrow down your results

Get All Characters

GET /api/characters

Get Character by ID

GET /api/characters/:id

Get Random Character

GET /api/characters/random

Get Character Quotes

GET /api/characters/:id/quotes

Available Filters

Parameter Type Description
name string Filter by character name
status string Filter by status (Alive, Deceased, Unknown)
gender string Filter by gender
occupation string Filter by occupation
season number Filter by season appearance

Example

GET /api/characters?status=Alive&gender=Female

Get All Creatures

GET /api/creatures

Get Creature by ID

GET /api/creatures/:id

Get Random Creature

GET /api/creatures/random

Available Filters

Parameter Type Description
name string Filter by creature name
origin string Filter by origin
threat_level string Filter by threat level (Low, Medium, High, Extreme)
status string Filter by status

Get All Episodes

GET /api/episodes

Get Episode by ID

GET /api/episodes/:id

Get Episodes by Season

GET /api/seasons/:season/episodes

Get Random Episode

GET /api/episodes/random

Available Filters

Parameter Type Description
season number Filter by season (1-4)
title string Filter by episode title
directed_by string Filter by director

Get All Locations

GET /api/locations

Get Location by ID

GET /api/locations/:id

Get Random Location

GET /api/locations/random

Available Filters

Parameter Type Description

status string Filter by status
