## ğŸš€ Getting Started

### Prerequisites

*   Node.js (v14 or higher)
*   npm (v6 or higher)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/JoseAlvarezDev/Stranger-Things_API.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Stranger-Things_API
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the server:
    ```bash
    npm run dev
    ```
5.  Open your browser at `http://localhost:3000`.
